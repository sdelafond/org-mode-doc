9 Capture - Refile - Archive

An important part of any organization system is the ability to quickly capture new ideas and tasks, and to associate reference material with them. Org defines a capture process to create tasks. Once in the system, tasks and projects need to be moved around. Moving completed project trees to an archive file keeps the system compact and fast.
